## 🧠 Descripción

Este portfolio fue desarrollado como parte de un proceso de formación en desarrollo web, aplicando buenas prácticas de maquetación, diseño responsive y organización del código.

El objetivo principal es mostrar:

- Información profesional
- Tecnologías dominadas
- Proyectos destacados
- Métodos de contacto

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox + Grid)
- Responsive Design (Media Queries)
- Google Fonts
- ChatGPT (Diseño de los efectos visuales)

---

## 🎨 Características principales

- ✅ Diseño moderno y minimalista  
- ✅ Layout responsive (mobile-first)  
- ✅ Secciones organizadas semánticamente  
- ✅ Tarjetas de proyectos con enlaces funcionales  
- ✅ Efectos visuales (glow / blur con CSS)  
- ✅ Footer con redes sociales interactivas  

---

## 📱 Responsive Design

El proyecto está optimizado para diferentes tamaños de pantalla:

- 📱 Mobile: 320px – 767px  
- 💻 Desktop: +768px  

Se utilizaron media queries para adaptar:

- Estructura del header  
- Grid de proyectos  
- Tamaños de texto  
- Espaciados y alineación
